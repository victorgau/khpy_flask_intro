#### Flask 的樣板語言

在 flask 的應用中，它可以使用 jinja2 樣板引擎，更有彈性的及更簡化的撰寫 HTML 檔。

在樣板中，主要的三種括號如下：

* {% ... %} for Statements
* {{ ... }} for Expressions to print to the template output
* {# ... #} for Comments not included in the template output
